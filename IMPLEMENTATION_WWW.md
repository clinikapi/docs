# Implementation: Landing Page (`apps/www`)

The marketing and landing page for ClinikAPI.

## Visual Principle
- **Premium Aesthetics**: High-end animations (Framer Motion), mesh gradients, and "Stripe-like" clean layouts.
- **Developer First**: Code front-and-center. Interactive widgets showing real SDK usage.
- **Copywriting**: Focused on compliance (HIPAA), speed, and developer simplicity.

## Engineering
- **Framework**: Next.js 15 (Static Export / Hybrid).
- **Icons**: Hugeicons (standard across Clinik ecosystem).
- **Typography**: Inter (Google Fonts).
- **Performance**: Edge-optimized images, 100/100 Lighthouse target.

## Key Sections
1. **Hero**: Visual representation of the "Healthcare OS".
2. **Interactive Code Window**: Tabbed interface showing SDK vs FHIR REST comparisons.
3. **Floating UI Widget**: Live demo of `@clinikehr/react` components.
4. **Feature Grid**: Detailed breakdown of compliance and automation.
5. **Developer CTA**: Terminal-style box with installation commands.
