# Imaging Studies

Imaging Studies (FHIR `ImagingStudy`) represent radiology and diagnostic imaging exams such as CT scans, MRIs, X-rays, and ultrasounds. ClinikAPI simplifies series to key fields — use the [FHIR passthrough](/guides/raw-fhir) for DICOM instance-level access.

## Record a CT Scan

```ts
const { data: study } = await clinik.imagingStudies.create({
  status: 'available',
  patientId: 'pt_jane123',
  encounterId: 'enc_er456',
  modality: ['CT'],
  started: '2024-03-15T10:00:00Z',
  referrerId: 'prac_drSmith',
  numberOfSeries: 2,
  numberOfInstances: 120,
  procedureCode: ['CT chest with contrast'],
  locationId: 'loc_radiology',
  description: 'CT Chest — rule out pulmonary embolism',
  reasonCode: ['Shortness of breath', 'Chest pain'],
  series: [
    {
      uid: '1.2.840.113619.2.55.3.604688119.969.1364202859.234',
      number: 1,
      modality: 'CT',
      description: 'Axial images with contrast',
      numberOfInstances: 80,
      bodySite: 'Chest',
    },
    {
      uid: '1.2.840.113619.2.55.3.604688119.969.1364202859.235',
      number: 2,
      modality: 'CT',
      description: 'Coronal reformats',
      numberOfInstances: 40,
      bodySite: 'Chest',
    },
  ],
  basedOn: ['ServiceRequest/sr_ctChest789'],
});
```

## Search Imaging Studies

```ts
// All studies for a patient
const { data } = await clinik.imagingStudies.search({
  patientId: 'pt_jane123',
});

// Filter by modality
const { data: ctScans } = await clinik.imagingStudies.search({
  patientId: 'pt_jane123',
  modality: 'CT',
});

// Filter by date range
const { data: recent } = await clinik.imagingStudies.search({
  dateFrom: '2024-03-01',
  dateTo: '2024-03-31',
  status: 'available',
});
```

## Common Modalities

| Code | Description |
|------|-------------|
| `CT` | Computed Tomography |
| `MR` | Magnetic Resonance |
| `US` | Ultrasound |
| `XR` | X-Ray (Radiography) |
| `NM` | Nuclear Medicine |
| `PT` | PET Scan |
| `MG` | Mammography |
| `RF` | Fluoroscopy |
