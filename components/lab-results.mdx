# LabResultsWidget

Displays lab reports (DiagnosticReport) with individual result values and interpretation flags.

## Usage

```tsx
import { LabResultsWidget } from '@clinikapi/react';

<LabResultsWidget
  proxyUrl={`/api/clinik/patients/${patientId}/labs`}
  onError={(err) => console.error(err)}
/>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `proxyUrl` | `string` | Yes | Backend proxy URL returning lab reports |
| `className` | `string` | No | CSS class name |
| `onError` | `(error: Error) => void` | No | Error callback |

## Proxy Endpoint

```ts
export async function GET(req: Request, { params }: { params: { id: string } }) {
  const { data } = await clinik.labs.search({
    patientId: params.id,
    sort: '-date',
    count: 20,
  });
  return Response.json(data);
}
```

## Features

- Report list with status badges (preliminary, final, amended)
- Individual result values with reference ranges
- Abnormal value highlighting (H, L, HH, LL flags)
- Conclusion text display
- Attached document links
