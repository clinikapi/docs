# NoteEditor

A clinical note editor supporting multiple note types with rich text input.

## Usage

```tsx
import { NoteEditor } from '@clinikapi/react';

<NoteEditor
  proxyUrl="/api/clinik/notes"
  onError={(err) => console.error(err)}
/>
```

## Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `proxyUrl` | `string` | Yes | Backend proxy URL for note CRUD |
| `className` | `string` | No | CSS class name |
| `onError` | `(error: Error) => void` | No | Error callback |

## Features

- Note type selector (progress note, discharge summary, consultation, H&P, etc.)
- Rich text content area
- Document status management (preliminary → final → amended)
- Category tagging
